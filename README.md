# AÑADIR UN NUEVO SIMULADOR
Actualmente los principales simuladores que sería interesante añadir serían en orden de prioridad:
-Concurrente (2º curso): es similar a operativos y se puede usar el mismo simulador, sería solamente crear un txt con preguntas de examenes anteriores.
-Lógica (2º curso): existe el santo grial pero tenerlas en simulador sería mejor para memorizarlas.
-Redes 2 (2º curso) y Distribuidos (3º curso), ambas asignaturas tienen todos los examenes anteriores publicados, sería de interés poder practicar intuitivamente con el simulador.
